# TON Connect Test Project

Bu proje `@blazium/ton-connect-mobile` paketini test etmek için oluşturulmuş bir örnek Expo uygulamasıdır.

## Kurulum

```bash
npm install
```

## Çalıştırma

```bash
# Expo development server'ı başlat
npm start

# Android'de çalıştır
npm run android

# iOS'ta çalıştır
npm run ios

# Web'de çalıştır
npm run web
```

## Özellikler

- ✅ Wallet bağlantısı
- ✅ Transaction gönderme
- ✅ Wallet durumu takibi
- ✅ Bağlantı kesme

## Notlar

- `react-native-get-random-values` import edilmeli (App.tsx'in en üstünde)
- Manifest URL'i gerçek bir URL olmalı
- Deep link scheme'i (`testapp`) app.json'da tanımlı
